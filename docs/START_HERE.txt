Welcome! Here's how to play the game:

OPTION 1: Direct Browser Open (Easiest)
----------------------------------------
1. Double-click "index.html"
2. Game opens in your default browser
3. Click "Start Your Journey"
4. Play through 13 eras of disruption!

OPTION 2: Local Server (Better for testing)
-------------------------------------------
1. Open terminal/command prompt
2. Navigate to this folder:
   cd C:\Users\Ariel\source\PivotOrDie\gameV1

3. Start a local server:

   Python:
   python -m http.server 8000

   Node.js:
   npx serve

   PHP:
   php -S localhost:8000

4. Open browser to: http://localhost:8000
5. Click "Start Your Journey"
